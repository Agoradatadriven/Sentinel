SENTINEL LOGO — paste your logo here
====================================

Drop your logo file into THIS folder and name it exactly one of:

    logo.svg      (preferred — stays sharp at any size)
  or
    logo.png      (use a transparent background for best results)

It automatically appears in:
  - the sidebar (every page after login)
  - the login screen
  - the kiosk and phone-scanner screens

Recommended: a horizontal / landscape logo, roughly 150 x 40 proportions,
transparent background. If no file is here, the built-in AGORA mark shows.

AFTER you add the file
----------------------
  * Locally: just refresh the page — it shows immediately.
  * Live site: redeploy from the "sentinel" folder in PowerShell:
        .\deploy\deploy.ps1

Optional — browser tab icon (favicon):
  Replace the file one level up:  ../favicon.svg
  (a small SQUARE version of your mark).

Note: the small amber "SENTINEL" pill next to the logo is separate text.
If your logo already includes the word "Sentinel" and you want the pill
removed, just ask.
